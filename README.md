# ChaoJia - AI 聊天聚合器

## 🎯 项目目标

一个简单的 Chrome 浏览器扩展，让用户在一个聊天窗口里同时和 ChatGPT、Gemini 对话。

---

### 打开方式

点击浏览器工具栏的扩展图标 → 弹出聊天窗口（popup 或新标签页）

## 🛠️ 技术方案

### 架构

```
chaojia/
├── manifest.json          ← Chrome 扩展配置
├── popup.html             ← 点击图标弹出的页面（或用新标签页）
├── popup.js               ← 聊天界面逻辑
├── popup.css              ← 聊天界面样式
├── background.js          ← Service Worker，管理 iframe 和消息路由
├── content-scripts/
│   ├── chatgpt.js         ← ChatGPT 网页适配器
│   └── gemini.js          ← Gemini 网页适配器
└── icons/                 ← 扩展图标
```

### 关键技术点

1. **iframe 加载 AI 网页**
   - 使用 `<iframe>` 加载 ChatGPT 和 Gemini
   - 通过 `declarativeNetRequest` 修改响应头，移除 X-Frame-Options 限制

2. **Content Script 注入**
   - 向 iframe 中的 AI 网页注入 content script
   - 读取回复内容（通过 DOM 选择器）
   - 发送消息（模拟输入和点击发送按钮）

3. **消息路由**
   - 用户输入 → 同时发送给两个 content script
   - 两个 content script 各自读取回复
   - 回复发送回聊天界面显示

## 📋 依赖说明

- **ChatGPT 登录态**：用户需要先在浏览器里登录 ChatGPT
- **Gemini 登录态**：用户需要先在浏览器里登录 Gemini
- **Chrome 浏览器**：仅支持 Chrome / Chromium 系浏览器

## ⚠️ 已知风险

1. **网站反爬**：AI 网站可能检测到 iframe 加载并显示验证
2. **DOM 变化**：网站改版后可能需要更新适配器
3. **登录态过期**：需要用户重新登录

## 📝 版本规划

- **v0.2**：优化体验，添加加载状态、错误处理
